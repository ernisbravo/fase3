Enlaces

El atributo target permite especificar el contexto en el que se resuelve el enlace. Los valores posibles son: _blank: el enlace se abre en una nueva pestaña _self: el enlace se abre en el mismo elemento que contiene el enlace (puede ser la ventana completa o un elemento contenedor como
